# WaterAgentLab

WaterAgentLab is a Python project for simulating water-resource allocation during drought.

The first MVP focuses on a simple, testable simulation engine:
- configurable drought scenarios
- stakeholder water demands
- baseline water allocation
- validation
- fairness and conflict metrics

Future versions will add:
- rule-based agents
- LLM-powered stakeholder negotiation
- experiment tracking
- model inspection
- French public drought and hydrometry data
